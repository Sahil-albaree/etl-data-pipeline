# 📊 ETL Data Pipeline

## 🚀 Overview

This project is a production-style ETL pipeline built using Python and MySQL. It automates data extraction, transformation, validation, and loading with scheduling and logging.

---

## 🛠️ Features

* Data Extraction from CSV
* Data Transformation & Cleaning
* Data Validation
* MySQL Data Loading
* Logging & Error Handling
* Automated Scheduling

---

## ⚙️ Tech Stack

* Python
* Pandas
* MySQL
* Schedule

---

## ▶️ How to Run

1. Clone the repo

2. Install dependencies
   pip install -r requirements.txt

3. Set environment variable
   DB_PASS=your_password

4. Run
   python scheduler.py
